h1. Amazon S3 Library for CodeIgniter

Easily integrate your CI applications to Amazon's Simple Storage Solution with this library.

This library is (hardly) a port of "Donovan SchÃ¶nknecht's S3 PHP Class":http://undesigned.org.za/2007/10/22/amazon-s3-php-class.

h2. Setup

# Open S3.php and edit lines 44 and 45 with your access and secret keys correspondingly.
# Drop S3.php in your applications/library folder.

h2. Example Usage

@// Load Library@
@$this->load->library('s3');@

@// Create a Bucket@
@var_dump($this->s3->putBucket('My-Bucket', $this->s3->ACL_PUBLIC_READ));@

@// List Buckets@
@VAR_DUMP($this->s3->listBuckets());@
